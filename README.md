## Build Setup

``` bash
# 安装模块依赖
npm install

# 启动开发环境项目
npm run dev

# 生成产品发布项目包
npm run build

# 生产产品包和分析报告
npm run build --report

git init //把这个目录变成Git可以管理的仓库
git add . //提交所有到本地仓
git commit -m "first commit" //把文件提交到仓库
git remote add origin url //关联远程仓库
git pull --rebase origin master //更新
git push -u origin master //把本地库的所有内容推送到远程库上
```
